# Drop-in images

Everything works without these (elegant gradient/initial fallbacks stand in).
Add a file and it appears automatically on next reload — no code changes.

> ⚠️ Game covers and anime posters are copyrighted by their studios. Fine for a
> personal hobby page; if you publish widely, use art you have the right to use.

## You (required to personalize)
| File | Where it shows |
|------|----------------|
| `profile.jpg` | Homepage hero avatar, About panel, "off the clock" section |

## Section backgrounds (optional)
| File | Where |
|------|-------|
| `poe2-warrior.jpg` | Projects (PoE2) hero background |
| `img/offduty-bg.jpg` | Off-Duty hero background (behind the crystal scene) |

- `anime/lord-of-mysteries.jpg`
- `anime/fullmetal-alchemist-brotherhood.jpg`

- `anime/solo-leveling.jpg`

The slug is the title lowercased with non-alphanumerics turned into hyphens
(e.g. "Counter-Strike 2" → `counter-strike-2`).
